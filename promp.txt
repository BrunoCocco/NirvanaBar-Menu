Estamos construyendo el proyecto freelance llamado "NIRVANA BAR MENU".

Stack:

* Next.js (App Router)
* React
* Supabase (database + auth + API)
* Tailwind
* Deploy en Vercel.

Arquitectura del proyecto:

Tabla menu_items:
id
name
description
price
category
available

Tabla specials:
id
type (plato_dia, menu_semana, vino_casa)
title
description
price
start_time
end_time
updated_at

El dashboard permitirá editar:

* plato del día
* menú de la semana
* vino de la casa
  con precio y horario.

El menú público mostrará los datos desde Supabase mediante queries del cliente supabase-js.
